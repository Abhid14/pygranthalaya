Welcome to Pygranthalaya

Pygranthalaya is an open source library management system providing friendly User - interface built with Open source software. Built on Python and Electron powered by EEL.js pip package and sqlite has been used to provide simple and zero configuration database built for speed. A project by Abhishek Das & team for more information about this software visit https://github.com/Abhid14/pygranthalaya .
